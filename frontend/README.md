# À La Carte

Aplicação mobile-first de receitas, desenvolvida inicialmente com Angular e Tailwind CSS.

## Objetivo

O À La Carte é um aplicativo de receitas pensado para uso mobile, permitindo organizar receitas, ingredientes, quantidades, checklist editável e modo de preparo.

A proposta visual busca transmitir uma experiência amigável, confortável e acolhedora, mantendo um nível de organização que também possa ser útil para pessoas da área de cozinha.

## Estrutura do projeto

- `frontend/` - aplicação web mobile-first em Angular + Tailwind CSS
- `backend/` - futura API em Java Spring Boot

## Tecnologias atuais

- Angular
- TypeScript
- Tailwind CSS
- HTML
- CSS
